PythonExtra differences from CPython
====================================

PythonExtra implements Python 3.4 and some select features of Python 3.5 and
above.  The sections below describe the current status of these features.

For the features of Python that are implemented by PythonExtra, there are
sometimes differences in their behaviour compared to standard Python.  The
operations listed in the sections below produce conflicting results in
PythonExtra when compared to standard Python.
